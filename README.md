# MIDI Arduino UNO USB Host Shield
Project yang menggunakan Arduino UNO + USB Host Shield untuk keperluan mengirim MIDI Message (PC dan CC)
Tujuannya untuk membuat MIDI Switch Controller melalui protokol USB Host.

Project ini menggunakan Sonicake Pocket Master untuk testing, 
namun project ini bisa digunakan untuk SEMUA EFEK GITAR yang menggunakan protokol USB HOST,
misal : Valeton GP5, GP50, GP150, GP180, 
Sonicake Pocket Master dan Smart Box, 
Semua UAFX 2.0 Pedal, 
Nux Amp Academy, mg300, MG-30, 
Zoom multi-effects pedal kaya MS-50G, MS-70CDR, dan G1 Four, 
dan lain sebagainya.

Project ini masih dalam tahap pengembangan.
